# 🚗 ParkSense

An intelligent, data-driven **Smart Parking Lot Management System** designed to optimize parking space utilization using real-time occupancy tracking, dynamic pricing, and analytics.

This project simulates a real-world urban parking infrastructure system that maximizes revenue, reduces congestion, and improves user experience through automation and predictive insights.

---

## 📌 Overview

Urban parking systems often suffer from:
- Inefficient fixed pricing
- Overcrowding during peak hours
- Underutilization during low-demand periods
- Lack of real-time availability visibility

This system addresses these challenges with:
- Real-time slot monitoring  
- Demand-based dynamic pricing  
- Revenue analytics dashboard  
- Data-driven decision making  

---

## 🎯 Key Features

### 🔹 Real-Time Slot Management
- Track available and occupied parking slots
- Log vehicle entry and exit
- Automatic occupancy update
